## 🎓 经验总结

### 成功经验
1. **MVP 优先**：先实现核心功能，再逐步迭代
2. **用户体验**：移动端优先，交互流畅
3. **AI 集成**：提示词设计是关键
4. **数据模型**：清晰的数据结构便于扩展
5. **文档完善**：详细的文档降低使用门槛

### 遇到的挑战
1. **CORS 限制**：千问 API 不支持前端直接调用
   - 解决：Vercel serverless 后端代理
2. **语音识别兼容性**：iOS Safari 支持有限
   - 解决：提供降级方案
3. **PWA 限制**：无法实现单 HTML 文件
   - 解决：3 文件架构
