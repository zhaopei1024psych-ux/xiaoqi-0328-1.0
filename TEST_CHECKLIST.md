## 测试场景

### 场景 1：新用户首次使用
1. 打开应用
2. 看到 API Key 配置提示
3. 进入设置页面
4. 输入 API Key 并保存
5. 返回待办页面

### 场景 2：AI 解析任务
1. 在输入框输入："明天上午10点开会，下周五交论文"
2. 点击"AI 解析任务"
3. 等待解析完成
4. 查看是否创建了 2 个待办
5. 检查时间是否正确

### 场景 3：添加课程
1. 切换到"课程表"标签
2. 点击"+ 添加课程"
3. 填写课程信息
4. 保存
5. 查看课程是否显示在对应星期

### 场景 4：设置校历
1. 切换到"日历"标签
2. 设置学期起止日期
3. 添加节假日（如：清明节 4月4日-6日）
4. 保存
5. 查看节假日列表

### 场景 5：冲突检测
1. 添加一个待办：明天下午3点考试
2. 再添加一个待办：明天下午4点交作业
3. 查看是否弹出冲突提醒

### 场景 6：记忆系统
1. 进入设置 → AI 记忆管理
2. 添加记忆：key=group_default_people, value=3, description=小组作业默认人数
3. 保存
4. 在待办页面输入："小组作业，下周一交"
5. 查看 AI 是否根据记忆处理

### 场景 7：日历导出
1. 确保有未完成的待办
2. 切换到"日历"标签
3. 点击"导出待办到日历"
4. 下载 .ics 文件
5. 用系统日历打开，查看是否正确导入

### 场景 8：数据持久化
1. 添加几个待办、课程
2. 刷新页面
3. 查看数据是否保留

### 场景 9：主题切换
1. 进入设置
2. 开启深色模式
3. 查看界面是否变为深色
4. 刷新页面，查看主题是否保留

### 场景 10：语音输入
1. 点击麦克风图标
2. 说："明天上午9点开会"
3. 查看是否转为文字
4. 点击 AI 解析

## 已知限制

1. **语音识别**：
   - iOS Safari 支持有限
   - Firefox 不支持
   - 需要麦克风权限

2. **API 调用**：
   - 需要配置 API Key
   - 直接调用可能因 CORS 失败（建议使用 Vercel 部署）

3. **截图识别**：
   - 功能提示"即将推出"
   - 需要千问视觉模型

4. **PWA**：
   - 需要 HTTPS 环境
   - iOS 需要手动添加到主屏幕
